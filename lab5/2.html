Twitter Bootstrap

Свободный набор инструментов для создание сайтов и веб-приложений

Включает в себя HTML и CSS шаблоны оформления для типографии веб-форм
, меток, блоков, навигации и прочих компонентов веб-интерфейсов включая JavaScript расширения
Сетки - заранее заданные размеры колонок, которые можно сразу же
использовать, например ширина колонки 140px относится к классу span2
который можно использовать в CSS описаниии документа.
Типографика - Описание шрифтов, определение некоторых классов для
шрифтов, таких как код, цитаты и т.п.
Медиа - Представляет некоторое управление изображениями и видео.
